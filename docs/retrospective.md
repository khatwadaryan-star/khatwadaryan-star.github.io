# Project Reflection
## What Went Well?
- Breaking tasks into phases under the WBS improved organization.
- Using a single CSS file ensured consistent design.
## What Went Wrong?
- Many revisions were required for certain files due to errors.
- CSS alignment issues took longer than desired.
## What Would You Do Differently?
- Test responsiveness earlier in the development phase.
- Make sure I am clear of error before committing changes.
